Core HTTP Concepts

Understanding these terms helps explain how the web works under the hood.

TermWhat It Means
HTTP vs HTTPSHTTPS = HTTP + encryption (TLS)
Persistent ConnectionReusing a TCP connection for multiple requests (HTTP/1.1)
MultiplexingSending many requests/responses at once over one connection (HTTP/2)
QUICA modern transport protocol using UDP instead of TCP (used in HTTP/3)
TLSEncryption layer that secures data in transit
Stateless ProtocolEach HTTP request is independent—no memory of past requests
Server PushServer sends files the browser hasn’t asked for yet (HTTP/2 feature)
